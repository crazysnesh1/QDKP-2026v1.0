QDKPv2 - За основу взял русифицированную версию @Joneil7
- Добавил:
  - Сокрытие альтов
  - Поиск по нику(При вводе мейна - видны все привязанные твины)
  - Счётчик участников гильдии(При сокрытие альтов отображает число мейнов, при поиске по нику мейна показывает количество привязанных персонажей)
  - Добавил колонку и кнопку обновления гильдейских заметок
  - Добавил колонку и кнопку назначения Ролей для авто-системы наград за Роли (Сумма наград за роли редактируется в QDKP_V2\RoleBonus.lua)
  - Добавил систему Авто начисления ДКП за планки ДПС на определенных боссах (Сумма наград за ДПС редактируется в QDKP_V2\DpsBonus.lua)
  - Добавил 2 кнопки "макроса" для старта таймеров в обычке для доп ДПК (Содержание "макроса" редактируется в QDKP2_GUI\ICCTimers.lua)
  - Добавил кнопку для анонса в /rw или просто рейд чат(Если нет асиста) пугов(Не состоящих или не привязанных персонажей к ГИ) в рейде
  - Расширил возможности чека закрытия энкаунтера(Для авто начисления за боссов):
    - ЦЛК - Работает полностью
    - РС - Работает, при условии что хост сессии в свету, во тьме не проходит, нужен фикс
    - ИВК - Работает полностью
    - Ульдуар - Работает полностью (Пока аддон не определяет в ХМ или ОБ убит босс, поэтому награда регается за сложность рейда "ОБ")
    - Накс - Работает полностью (За исключением ХМ мода на Циркуле)
  - Добавил возможнсть импорта/экспорта списка не гильдейских персонажей, между офицерами внутри игры!
  - Ну и еще по мелочи правок, для упрощения администрирования гильдей офицерам.
